# Theme

This is a custom WordPress theme.

# Installation
1. Clone the repository into your /wp-content/themes/ directory:
git clone https://github.com/KatMelnik/specialist.git
2. Activate the theme in the WordPress Dashboard.
3. Import ACF Fields: import the acf-export-2026-02-06.json file (found in the root directory) into your ACF plugin to set up all necessary fields.
4. Ensure you have some "Specialist Categories" created and assigned to posts.
